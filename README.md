# Daniel Kivatinos

Serial entrepreneur building at the intersection of AI and finance. Previously co-founded DrChrono (YC W11) — the first iPad-native medical records system — impacting millions of patients and providers. Now building JustPaid, an autonomous AI billing platform for B2B.
